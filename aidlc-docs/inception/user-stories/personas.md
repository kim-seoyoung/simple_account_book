# User Personas

## Persona 1: 가계부 기록자

| 항목 | 내용 |
|------|------|
| 이름 | 김지영 (대표 페르소나) |
| 역할 | 가족 소비 기록 및 통계 확인 |
| 목표 | 가족의 월별 소비를 체계적으로 파악하고 절약 포인트를 찾기 |
| 사용 패턴 | 하루/주 단위로 소비 내역을 모아서 일괄 입력 |
| 주요 관심사 | 카테고리별 소비 비율, 월별 소비 추이 변화 |
| 기술 수준 | 웹 브라우저 기본 사용 가능 |
| Pain Point | 소비 내역을 빠르게 여러 건 입력하고 싶음, 한눈에 소비 패턴을 파악하고 싶음 |

---

**Note**: 인증 없이 단일 접근 방식이며, "사용자"는 소비 내역에 누가 썼는지 표시하는 라벨 역할만 수행. 모든 가족 구성원이 동일한 화면과 기능을 사용.
