# 가계부 요구사항 정의서

## 1. 프로젝트 개요

### 1.1 서비스 비전
가정에서 소비하는 금액을 종합적으로 파악가능하도록 정리해주는 플랫폼

### 1.2 핵심 가치 제안
- 월별 소비 내역을 카드별, 카테고리별, 사용자별로 체계적으로 기록
- 직관적이고 사용하기 쉬운 인터페이스
- 월별/연간 통계를 통한 소비 패턴 파악 및 비교

---

## 2. 서비스 구성

### 2.1 주요 구성 요소
- **사용자 인터페이스**: 브라우저에서 동작하는 웹UI 방식
- **서버 시스템**: 소비 기록 처리 및 데이터 관리
- **데이터 저장소**: 소비 내역, 카드 정보, 카테고리, 통계 정보 저장

---

## 3. 핵심 기능 요구사항

### 3.1 소비 기록 관리

#### 3.1.1 소비 내역 등록
**목적**: 사용자가 일상 소비를 빠르게 기록

**기능 요구사항**:
- 소비 항목 입력
  - 날짜
  - 금액
  - 사용자 (선택)
  - 사용 카드 (선택)
  - 카테고리 (선택)
  - 메모 (선택)
- 필수 필드: 날짜, 금액
- 금액은 양수만 허용

#### 3.1.2 소비 내역 조회
**목적**: 월별로 소비 내역을 확인

**기능 요구사항**:
- 월 단위로 소비 내역 목록 표시
- 각 내역에 날짜, 금액, 사용자, 카드, 카테고리, 메모 표시
- 카테고리별 필터링
- 카드별 필터링
- 사용자별 필터링
- 날짜순 정렬 (최신순 기본)

#### 3.1.3 소비 내역 수정/삭제
**목적**: 잘못 입력한 내역을 수정하거나 삭제

**기능 요구사항**:
- 기존 소비 내역 수정
- 소비 내역 삭제 (확인 팝업 표시)

### 3.2 카드 관리

#### 3.2.1 카드 등록/관리
**목적**: 사용하는 카드 목록을 관리

**기능 요구사항**:
- 카드 이름 등록
- 카드 목록 조회
- 카드 수정/삭제
- 삭제 시 해당 카드로 기록된 내역은 "카드 없음"으로 변경

### 3.3 카테고리 관리

#### 3.3.1 카테고리 등록/관리
**목적**: 소비 분류를 위한 카테고리 관리

**기능 요구사항**:
- 카테고리 이름 등록
- 기본 카테고리 제공 (식비, 교통, 쇼핑, 주거, 통신, 의료, 여가, 기타)
- 카테고리 추가/수정/삭제
- 삭제 시 해당 카테고리의 내역은 "기타"로 변경

### 3.4 사용자 관리

#### 3.4.1 사용자 등록/관리
**목적**: 가계부를 함께 사용하는 가족 구성원을 관리

**기능 요구사항**:
- 사용자 이름 등록
- 사용자 목록 조회
- 사용자 수정/삭제
- 삭제 시 해당 사용자의 소비 내역은 "사용자 없음"으로 변경

### 3.5 월별 통계

#### 3.5.1 월별 소비 통계
**목적**: 월 단위 소비 패턴을 한눈에 파악

**기능 요구사항**:
- 월 선택 기능
- **카테고리별 통계**:
  - 카테고리별 총 소비 금액
  - 카테고리별 비율 (%)
  - 차트 시각화 (파이 차트 또는 바 차트)
- **카드별 통계**:
  - 카드별 총 소비 금액
  - 카드별 비율 (%)
- **사용자별 통계**:
  - 사용자별 총 소비 금액
  - 사용자별 비율 (%)
- **월 총계**:
  - 해당 월 총 소비 금액
- **전월 대비**:
  - 전월 대비 증감액 및 증감률 표시

#### 3.5.2 연간 통계
**목적**: 12개월 소비 추이를 한눈에 비교하여 연간 소비 패턴 파악

**기능 요구사항**:
- 연도 선택 기능
- **월별 소비 추이**:
  - 1월~12월 월별 총 소비 금액 표시
  - 라인 차트 또는 바 차트로 시각화
- **카테고리별 연간 추이**:
  - 카테고리별 월별 소비 금액 비교
- **카드별 연간 추이**:
  - 카드별 월별 소비 금액 비교
- **사용자별 연간 추이**:
  - 사용자별 월별 소비 금액 비교
- **연간 총계**:
  - 해당 연도 총 소비 금액
- **전년 대비**:
  - 전년 동월 대비 증감 표시
  - 전년 총액 대비 증감률 표시

---

## 4. MVP 개발 범위

### 핵심 기능 (필수)
**목표**: 월별 소비 기록 및 통계 확인

- 소비 내역 등록/조회/수정/삭제
- 카드 등록/관리
- 카테고리 등록/관리
- 사용자 등록/관리
- 월별 통계 (카테고리별, 카드별, 사용자별 합계 및 비율)
- 연간 통계 (월별 추이 비교, 전년 대비)

---

## 부록

### A. 용어 정의
- **MVP**: Minimum Viable Product (최소 기능 제품)
- **카테고리**: 소비 항목의 분류 (식비, 교통 등)
- **월별 통계**: 특정 월의 소비 데이터를 집계한 요약 정보
- **연간 통계**: 12개월 소비 데이터를 월별로 비교한 요약 정보
- **사용자**: 가계부를 함께 사용하는 가족 구성원
